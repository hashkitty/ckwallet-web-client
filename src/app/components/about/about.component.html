CK WALLET SEARCH powered by ckwallet-core

CK wallet search is a handy way to find kitties. It uses special query language (kitty query language or KQL). KQL is similar to query language used on CK site, but with some extra capabilities. Learn more below

Query by named trait

Just enter one or more trait name in the search box, joint by spaces or operators (OR/AND). Kitties with specified traits in dominant position will be returned

crazy crazy manx crazy or stunned Query by trait in recessive position

Just prefix name of the trait with a special prefix, indicating trait position: "r1:","r2:","r3:" (or "d:" for dominant, same as no prefix)

r1:crazy crazy r1:crazy r2:crazy r3:crazy crazy or r1:crazy Query by trait type (kai notation)

Enter trait type followed by semicolon and 1-4 trait values in kai. First value is for dominant trait, then first recessive and so on. So 'body:12' is equivalent to 'savannah r1:selkirk'. Full list of searchable trait types:

  • body
  • bodycolor
  • pattern
  • patterncolor
  • eyecolor
  • eyetype
  • accentcolor
  • mouth
  • wild
body:2222 eyeType:1 body:2 pattern:11 or pattern:21 Query by trait type (hex notation)

Same as kai but value is specified in hex notation. Value starts with '0x' (to indicate hex notation) followed by 1-8 hex digit. Each trait is specified as a byte in hex notation (1-2 hex digits), first 2 digits after semicolon is for dominant trait then first recessive and so on. So body:0x01010101 is equivalent to body:2222 in kai

body:0x01010101 eyeType:0x0 body:0x1 pattern:0x0000 or pattern:0x0302 Query by generation

Enter "gen" keyword followed by semicolon and value in decimal notation

gen:0 gen:999 crazy gen:0 Keywords

Enter "virgin" keyword to get only kitties who doesn't have children

virgin virgin gen:0 crazy gen:0 virgin Combining multiple statements

Query statements can be combined by spaces or operators "OR", "AND". Spaces are treated as implicit "AND" operator. Grouping statements with brackets is not currently allowed, so combine "AND" "OR" operators on your own rist.
